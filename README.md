# Alexa Custom Skill for WordCamp Osaka 2018

## Getting started

### Setup

```
$ git clone git@github.com:wcosaka2018/alexa-wordcamp-osaka.git
$ cd alexa-wordcamp-osaka/lambda/custom
$ npm i
```

### deploy

```
$ cd /PATH/TO/alexa-wordcamp-osaka
$ ask init
$ ask deploy
```


## Test

```
$ cd lambda/custom
$ npm test
```
